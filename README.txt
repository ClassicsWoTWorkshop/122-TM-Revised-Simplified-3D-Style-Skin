Thank you for downloading! To install:


1. Open the folder with the option you would like to have the remodel apply to ("OPTION 1 - Replaces 122 TM (stock)" or "OPTION 2 - Replaces 3D Style")

2. Open the folder named with the option of your choosing:

   - STAGE 1 [SIMPLEST] has no launcher pods, MG cover, or turret basket;

   - STAGE 2 [DEFAULT] has revised launcher pods + MG cover, no turret basket;

   - STAGE 3 [MOST COMPLEX] has added turret basket + revised launcher pods + MG cover;

3. Drag both .wotmod files from this folder to <Your WoT Directory\mods\<Game Version>.

(Example: C:\Games\World_of_Tanks\mods\1.12.0.0)
